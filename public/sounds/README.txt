Folder untuk file suara notifikasi SkyBox.

Taruh file audio (.mp3 / .ogg / .wav) di folder ini.
File otomatis bisa diakses lewat URL: /sounds/<namafile>

Nama file yang sudah didaftarkan di dropdown Settings (rename file kamu jadi ini):
  - custom1.mp3   -> tampil sebagai "Custom 1"
  - custom2.mp3   -> tampil sebagai "Custom 2"
  - custom3.mp3   -> tampil sebagai "Custom 3"

Mau nama lain / lebih banyak? Bilang ke Kiro untuk menambah entri di src/lib/sound.ts (TONES).

Catatan:
- Format paling aman & ringan: .mp3
- File besar sebaiknya dipangkas (notifikasi cukup < 1-2 detik)
